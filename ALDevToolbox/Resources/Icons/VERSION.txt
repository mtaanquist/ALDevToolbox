Only the icons referenced by the app live here. To add an icon:
  1. Copy `<name>.svg` from https://github.com/lucide-icons/lucide/tree/1.14.0/icons
     (or a newer pinned tag — bump this file when you do).
  2. Drop it into ALDevToolbox/Resources/Icons/ — the .csproj globs *.svg
     as embedded resources.
  3. Use it as `<Icon Name="<name>" Width="..." Height="..." />`.
