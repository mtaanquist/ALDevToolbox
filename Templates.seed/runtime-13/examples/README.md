# Example AL files (runtime 13)

Same conventions as `runtime-15/examples/README.md`. The element-type folder layout for runtime 13 means example folders here would correspond to `src/codeunits/`, `src/pages/`, etc.

The runtime 13 template currently has no folders with `example` fields set — every folder gets `.gitkeep` regardless of the include-examples toggle. If the team wants seeded examples for the older layout, add `example = "..."` entries to `Templates.seed/runtime-13/template.toml` and place the corresponding files under this directory.
